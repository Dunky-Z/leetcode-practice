编写一个程序判断给定的数是否为丑数。

丑数就是只包含质因数 2, 3, 5 的正整数。

示例 1:
```
输入: 6
输出: true
解释: 6 = 2 × 3
```
示例 2:
```
输入: 8
输出: true
解释: 8 = 2 × 2 × 2
```
示例 3:
```
输入: 14
输出: false 
解释: 14 不是丑数，因为它包含了另外一个质因数 7。
```
说明：
1 是丑数。
输入不会超过 32 位有符号整数的范围: [−231,  231 − 1]。

---
Solution 1:
循环不听除以这几个数，如果剩下1就是丑数

```cpp
class Solution {
public:
    bool isUgly(int num) 
    {
        while(num >= 2)
        {
            if(num % 2 == 0)
                num /= 2;
            else if(num % 3 == 0)
                num  /= 3;
            else if(num % 5 == 0)
                num /= 5;
            else
                return false;
        }
        return num == 1;
    }
};
```
